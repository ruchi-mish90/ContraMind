# ContraMind - The World's First Cognitive Gym

> "If you can't defend it, you don't believe it."

ContraMind is an AI-powered intellectual sparring partner designed to shatter echo chambers through adversarial but brilliant dialogue. It uses the Steel-Man protocol, real-time fallacy detection, and persona battles to forge steel-trap minds.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🎯 Features

- **Steel-Man Protocol**: Perfects your argument before destroying it
- **Fallacy Sniper**: Real-time detection of logical inconsistencies  
- **Persona Battles**: Debate Marx, Socrates, Nietzsche, Buddha
- **Pressure Gauge**: AI aggression adapts to your overconfidence
- **7-Day Challenge**: Ultimate test for Grandmaster status

## 🎯 User Journey

1. **Entry**: State a belief to challenge
2. **Cognitive Gym** (0-94%): Basic Socratic questioning
3. **Master Thinker** (95%+): Persona battles unlock
4. **Grandmaster** (100%): 7-day challenge system
5. **Completion**: Cognitive portrait generation

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS, Framer Motion
- **Backend**: Edge Runtime, Groq AI, TypeScript
- **State**: Zustand with persistence
- **Deployment**: Vercel (recommended)

## 🔒 Privacy

- No user data persistence
- Client-side state management
- Optional API key (has demo mode)
